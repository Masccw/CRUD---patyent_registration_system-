# Patient Registration System (CRUD)

## Problem
Hospitals manage a large volume of patient data daily. Manual or inconsistent systems can lead to duplicated records and data errors.

## Solution
A CRUD-based system developed in Python to manage patient records, enabling creation, retrieval, update, and deletion of data in a structured and secure way.

## Tech Stack
- Python
- Django
- SQLite
- HTML/CSS
- Git/GitHub

## Features
- Create, update, delete, and list patient records
- Basic data validation
- Relational data modeling

## Future Improvements
- Authentication and authorization
- Integration with appointment scheduling
- Deployment to cloud environment
